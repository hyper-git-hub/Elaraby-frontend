# IoP - Internet of Power
This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 1.7.4. Is working in Angular v5.


## Modules
1) App Module
2) IoP Module (lazy loaded)
3) Waterheater Module (lazy loaded)
4) Core Module.
5) Shared Module

## Deployment

#### Make a build
```
npm run att
```

#### View Documentation

```
npm run serve-docs
```
